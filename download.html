This page provides the link to the datasets. Our benchmark has two settings: 1) user-based separation and 2) time-based separation setting. The details of each set are reported in the paper. It should be noted that the Avocado (Personalized Email Subject Generation) dataset is not publicly available; however, we provided the code here and sample ids we used to generate the dataset. Follow the instructions to generate the dataset easily when you got acess to Avocado dataset.

Personalized Email Completion

This dataset is created using the private email collection dataset: Avocado Research Email Collection. The objective of this task is to complete the email for a given user, given an email subject, part of the email and user's previous email-subject pairs. For more information on this, refer to the longLAMP paper.

Personalized Abstract Generation

This dataset is generated using the Citation Network Dataset(V14). This task comprises of generating an abstract for a given author based on the author's previous research articles capturing the knowledge and stylistic properties. Abstract is generated using the title and the keywords of the article being written about.
